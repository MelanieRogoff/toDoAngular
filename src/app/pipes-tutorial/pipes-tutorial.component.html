Pipes Example:

Lowercase & Uppercase Pipes, Respectively:
{{ user.firstName | lowercase }}, {{ user.lastName | uppercase }}

-----------------------------------------------
Date Pipe with a timezone:
{{ user.DOB | date: 'MM/dd/YYYY: z' }}

-----------------------------------------------
Currency Pipe with Euros:
{{ user.salary | currency:'EUR' }}

-----------------------------------------------
Using JSON Pipe so we can see the full user object:
{{ user | json }}

-----------------------------------------------
